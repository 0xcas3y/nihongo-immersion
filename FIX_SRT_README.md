# fix_srt.py — SRT 字幕翻译补全工具

自动检测双语 SRT 字幕文件中缺失的中文翻译，使用 DeepL API 批量补全。

---

## 适用格式

专为以下格式设计的双语 SRT（奇数条目为日文，偶数条目为中文）：

```
1
00:00:00,080 --> 00:00:04,240
皆さん、こんにちは。

2
00:00:00,080 --> 00:00:04,240
大家好。

3
00:00:04,240 --> 00:00:09,040
最高に楽しかったです。

4
00:00:04,240 --> 00:00:09,040
 ...         ← 这种就是缺失翻译
```

---

## 使用方法

**第一步：把字幕文件和脚本放在同一个文件夹（或桌面）**

**第二步：终端运行**

```bash
# 先检查缺了多少条（不消耗 API 额度）
python3 fix_srt.py 字幕文件.srt --dry-run

# 执行翻译补全
python3 fix_srt.py 字幕文件.srt --key 你的DeepL_Key
```

输出文件自动保存为 `原文件名_fixed.srt`，原文件不会被修改。
